Upload Acronyms from YAML

YAML Format

Your YAML file should follow this format:

CIA:
  - full: Central Intelligence Agency
    category: government
NSA:
  - full: National Security Agency
    category: government
API:
  - full: Application Programming Interface
    category: technology
  - full: Advanced Persistent Infrastructure
    category: security

Each acronym can have multiple expansions. The category field is optional (defaults to "imported").
