Activity Log — Quick Reference
Live Updates

Entries are kept in sync via WebSockets. The Connected / Disconnected indicator in the toolbar shows the current connection state.

Changes made by other operators appear automatically — no refresh needed.

Navigation & Selection
  • Click a row to open the details pane.
  • Double-click a row to edit the entry.
  • ↑↓ arrow keys move between entries once one is selected.
  • Enter opens the edit form; tap Enter again to submit it, saving and closing the form.
  • Esc closes the details pane and deselects the entry, giving the table full width.
  • Ctrl+N creates a new entry.
Sorting
  • The default order is start date, newest first.
  • Click a column header to sort by that column.
  • Shift+click additional headers to add a secondary sort column.
  • Your sort preference is saved and restored on the next visit. Use Reset Sort to return to the default order.
Show / Hide Columns

Click Show/Hide Columns to toggle individual columns on or off. Your preferences are saved in your browser. Long-form fields (Command, Output, Description, Comments) are truncated in the table — select an entry to read the full content in the details pane.

Filtering

Type in the Filter Entries box to search across all fields, including fields not currently shown as columns. Filtering also includes the raw input and output text of any terminal recordings attached to entries. Results update as you type. Click or clear the box to remove the filter.

Details Pane & Deep Links

The details pane shows the full entry, evidence attachments, and a terminal recording player. Use the deep link button in the details pane to copy a URL that opens this log and scrolls directly to that entry — useful for sharing specific entries with coworkers.
